Elemental Positioning

Sep, 08, 2021


“Positioning… the Battle for Your Mind” - Al Ries

Position: static

A static element is not positioned in any special way; it is always positioned according to the normal flow of the page:

This div element has position: static;


Position: relative

A relative element is positioned relative to its normal position. This element is indented from where it would normally be:

This div element has position: relative;


Position: fixed

A fixed element always stays in the same place even if the page is scrolled. Look down at the bottom right hand corner of your screen to see the fixed position element:

This div element has position: fixed;


Position: absolute

An absolute element is positioned relative to the nearest positioned parent-container (unlike the fixed element which is fixed in relation to the viewport):

This div element has position: relative;
This div element has position: absolute;


I'm a sticky element and I'm sticking around here.

The sticky element sticks to the top of the page, when you reach its scroll position. Can be used to annoy people to signup for your fabulous newsletters.
